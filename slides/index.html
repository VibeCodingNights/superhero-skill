CONTEXT

What is superhero.com?

Social network

Posts + tips + comments — but every action is a transaction on æternity. No platform owns your content.

Token market

Every trend / community has a bonding-curve token. Buy a topic, ride momentum, sell when it fades.

On-chain identity

.chain names. 13+ chars = instant. ≤12 = auction.

Open API

api.superhero.com — no auth for reads. Trending feed is a single GET.

Speaker (90s):

  • Built on æternity (FATE VM, sealed-bid AENS auctions, oracle-friendly). Not Ethereum, not Solana — it's its own thing.
  • Bonding curve = price is a pure function of supply. price(n) is a view call. No order book, no AMM pool.

LESSONS, FAST

What I learned
so you don't have to

1.55% round-trip cost

Spread + buy premium + sell discount + gas. Anything below this TP is a guaranteed loss.

Affiliation is a trap

1% rebate ONLY via create_community. Direct buy reverts. Spammer found this; you won't farm it.

Trending score lags

Backend-aggregated. The leading indicator is the raw tx feed. Watch scripts/superhero-transactions.mjs.

Whales fire on schedule

Daily DCA at 17:00 UTC. Currently −3,584 AE underwater. Don't be the whale.

Concentration > diversification

On a 16-real-token market, holding 5 positions = holding noise. 1–2 conviction picks beats spray.

Hold longer than feels right

4-cycle exits ate every gain. 16 cycles lets you ride into the 17:00 UTC window.

Speaker (90s):

  • These are scars. First config: +3/-5/2-pos/4-cycle. R:R 1:4 needed 80% hit rate. Lost 5.41 AE on first cycle.
  • The platform is too small to "diversify". You will own a slice of garbage if you spread to 5.
  • Trending score is a great filter, terrible signal. Use raw tx feed + holder count for entries.
